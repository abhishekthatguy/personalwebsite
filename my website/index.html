ABout Me

I translate the UI/UX designs wireframes into actual code that will produce visual elements of the application with little or no discrepancies between actual designs and deliverables. I work as a bridge between graphical design and technical implementation, took an active role on both sides and define how the application looks as well as how it works.

I work with UX designer and backend-team to build responsive design web applications and manage projects with collaborating with mentors. I also assure that all user-input is validated before submitting it to back-end
